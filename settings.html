          サポート／使い方

          各機能の概要と使い方の説明です。不具合の報告や要望、質問等はGitHubのissuesにお願いします。

          なお、即時対応が難しいことはご了承ください。

          特定のチャンネルをリストに登録すると、そのチャンネルの動画を非表示にできます。

          リストへの登録方法は３通り

            ・チャンネル名横の×ボタンを押す

            ・アイコンをクリックし、ポップアップ内のリストにチャンネル名を追記する

            ・詳細設定の「非表示チャンネルリスト」からチャンネル名を入力して追加する

          リストは「編集」ボタンで名前の編集が、「削除」ボタンでリストからの削除が可能です。

          チャンネル名に特定のキーワードが含まれている場合、そのチャンネルを自動的に非表示にします。

          最大3つのキーワード全てを含む動画タイトルを非表示にします。単純な非表示の場合は、1つだけ登録してください。。

          キーワードセットは最大5000個・10文字以内で設定できます。

          リストは「編集」ボタンで名前の編集が、「削除」ボタンでリストからの削除が可能です。

          動画タイトルに特定のキーワードが含まれている場合、その動画を自動的に非表示にします。

          最大3つのキーワード全てを含む動画タイトルを非表示にします。単純な非表示の場合は、1つだけ登録してください。

          キーワードセットは最大5000個・30文字以内で設定できます。

          リストは「編集」ボタンで名前の編集が、「削除」ボタンでリストからの削除が可能です。

          例：

          • 「切り抜き」だけ非表示にしたい→「切り抜き」をセット
          • 「切り抜き」は見たいが、「切り抜き」と「YouTuber」の2つを含む動画は非表示にしたい→「切り抜き」と「YouTuber」をセット

          @から始まるID名をリストに登録すると、そのチャンネルのコメントを非表示にできます。

          リストへの登録方法は2通り

            ・コメントでID横の×ボタンを押す

            ・詳細設定の「非表示コメントユーザーリスト」から@つきのIDを入力して追加する

          リストは「編集」ボタンで名前の編集が、「削除」ボタンでリストからの削除が可能です。

          正規表現（リテラル記法）を使ってチャンネル名や動画タイトルをフィルタリングします。

          正規表現の例：

          
  非表示条件の例 / Examples of hide conditions

  ──────────────────────────────
  英語の文字を含む / Match if at least one English letter exists

  /.*[a-zA-Z].*/

  ──────────────────────────────
  数字のみ / Match if consists of only digits

  /^\d+$/

  ───────────────────────────────
  "keyword"を含む / Match if contains a specific word "keyword"

  /keyword/

  ──────────────────────────────
  "keyword1"を含むが、"keyword1" + "keyword2" は除外 / 
  Matches strings containing "keyword1" but not "keyword1" + "keyword2"
  
  /^(?=.*keyword1)(?!.*keyword2).*/

          エクスポート／インポート機能を使うことで、非表示リストやキーワードを保存・復元できます。

          「エクスポート」ボタンを押すと、現在の設定をファイルに保存できます。

          「インポート」ボタンを押すと、保存したファイルを読み込んで設定を復元できます。

          定期的なバックアップを推奨します。

          「ショート動画フィルター」ボタンを押して「有効」にすると、YouTubeのショート動画がすべて非表示になります。

          「ショート動画フィルター」ボタンを押して「無効」にすると、YouTubeのショート動画が再び表示されます（機能が無効の場合はボタンがグレー表示）。

          今後、プレイリスト等についても同様の機能を追加予定です。
